## 🚀 Quick Start

### Option 1: Vite (Recommended)
```bash
cd rtv33-vite
npm install
npm run dev
```

### Option 2: Create React App
```bash
cd rtv33-cra
npm install
npm start
```

### Option 3: Standalone
The `rtv33.jsx` file is a single React component. Import it into any existing React project:
```jsx
import RTV33 from './rtv33';
// render <RTV33 /> in your app
```

## 🌐 Deploy

### Vercel
1. Push to GitHub
2. Go to [vercel.com](https://vercel.com)
3. Import your repo
4. It auto-detects Vite — click Deploy

### Netlify
1. Push to GitHub
2. Go to [netlify.com](https://netlify.com)
3. Import your repo
4. Build command: `npm run build` | Publish: `dist`

### GitHub Pages
```bash
npm run build
# deploy the dist/ folder
```
